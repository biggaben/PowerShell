
### Conda Commands:

Alias     Command
-------   -------------------
ccon      CondaListShortcuts # List Conda shortcuts
cce       CondaCreateEnv # Create a new Conda environment
conda-update 'conda update -n base conda' # Update the base Conda installation
aenv      'conda activate' # Activate a Conda environment
lenv      'conda env list' # List all Conda environments
cue       CondaUpdateEnv # Update all packages in a specified environment
cre       CondaRemoveEnv # Delete a Conda environment
